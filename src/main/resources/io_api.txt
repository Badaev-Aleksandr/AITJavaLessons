int read()--> читает один байт из входного потока, если достигнут конец потока, возвращает -1
int read(byte[] b, int off, int len) --> читает до len из входного массива начиная с позиции off. Возвращает количество
считанных байтов или -1 если достигнут конец его потока
int read (byte[] b) --> считывает до b.length байтов из входного потока или возвращает -1 если достигнет конец потока
long skip(long n) --> пропускает n-байтов в этом потоке
int available () --> возвращает оценку количества байта, которые могут быть прочитаны этого входящего потока
void close()--> закрывает входной поток и освобождает все с тим связанные ресурсы
